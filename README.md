# 🍽️ Золота Ложка - Luxury Restaurant Website

## Описание проекта

Це преміум веб-сайт ресторану з сучасним дизайном, реальними зображеннями та крутими анімаціями.

## Особливості

### 🎨 Дизайн
- **Фонове зображення з Ken Burns ефектом** - плавна анімація масштабування та руху
- **Анімовані частинки** на фоні hero секції
- **3D трансформації** при наведенні на карточки
- **Parallax ефекти** для зображень
- **Glassmorphism** - ефект матового скла для елементів UI

### 🖼️ Реальні зображення з Unsplash
- Фонове зображення ресторану в hero секції
- Зображення вишуканих страв в секції About
- Професійні фото їжі в меню (стейки, морепродукти, паста, десерти)
- Фотографії шеф-кухарів та інтер'єру ресторану в галереї
- Атмосферне фото в секції контактів

### ✨ Анімації та ефекти
- **Ripple effect** на кнопках при кліку
- **Hover ефекти** з масштабуванням та обертанням зображень
- **Плавні переходи** між секціями
- **Rotating glow** навколо featured карточок
- **Smooth scrolling** по всьому сайту

### 🎯 Секції сайту
1. **Hero** - велика секція з фоновим зображенням, заголовком та статистикою
2. **About** - асиметричний layout з реальними фото та описом
3. **Menu** - masonry grid з карточками страв та цінами
4. **Gallery** - фотографії команди та атмосфери ресторану
5. **Contact** - split layout з формою бронювання
6. **Footer** - інформація про ресторан

## Як відкрити проект

### Варіант 1: Подвійний клік
Просто двічі клікніть на файл `index.html` - він відкриється у вашому браузері за замовчуванням.

### Варіант 2: Через браузер
1. Відкрийте ваш улюблений браузер (Chrome, Firefox, Edge)
2. Натисніть Ctrl+O (або File → Open)
3. Виберіть файл `index.html` з папки проекту

### Варіант 3: Live Server (для розробників)
Якщо у вас встановлено VS Code з розширенням Live Server:
1. Відкрийте папку проекту в VS Code
2. Клікніть правою кнопкою на `index.html`
3. Виберіть "Open with Live Server"
